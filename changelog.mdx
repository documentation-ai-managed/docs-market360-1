## Migration Guide

For v2.0.0 breaking changes, update your auth calls like this:

<CodeGroup tabs="JavaScript,Python">
````javascript
// Old
const response = await fetch('https://api.example.com/v1/listings', {
  headers: { 'API-Key': 'YOUR_API_KEY' }
});

// New
const response = await fetch('https://api.example.com/v2/listings', {
  headers: { 'Authorization': `Bearer ${YOUR_TOKEN}` }
});
````
````python
# Old
import requests
response = requests.get('https://api.example.com/v1/listings', headers={'API-Key': 'YOUR_API_KEY'})

# New
response = requests.get('https://api.example.com/v2/listings', headers={'Authorization': f'Bearer {YOUR_TOKEN}'})
````
</CodeGroup>

<ParamField path="token" param-type="string" required="true">

  Your new bearer token for v2 API access.

</ParamField>
