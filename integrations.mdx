## Overview

Integrate Market360 with popular third-party services to automate payments, enable social logins, streamline deliveries, and receive real-time notifications. These integrations help you build a more efficient marketplace tailored to Sierra Leone's needs. Start by obtaining your API key from the Market360 dashboard at `https://dashboard.market360.com/settings/api`.

<Callout kind="info">
  All integrations require a valid `YOUR_API_KEY` from your Market360 account. Store it securely and never expose it in client-side code.
</Callout>

## Payment Gateway Integrations

Connect popular payment processors to handle transactions securely.

<Tabs>
  <Tab title="Stripe" icon="dollar-sign">
    Create a Stripe account and link it to Market360.

    <Steps>
      <Step title="Get Stripe Keys" icon="key">
        Retrieve your publishable and secret keys from the Stripe dashboard.
      </Step>
      <Step title="Configure in Market360" icon="settings">
        Add your keys in the dashboard under Payments > Stripe.
      </Step>
      <Step title="Test Checkout">
        ```javascript
        const stripe = Stripe('pk_test_YOUR_STRIPE_PUBLISHABLE_KEY');
        const {error} = await stripe.redirectToCheckout({
          lineItems: [{price: 'price_12345', quantity: 1}],
          mode: 'payment',
          successUrl: 'https://your-site.com/success',
          cancelUrl: 'https://your-site.com/cancel',
        });
        ```
      </Step>
    </Steps>
  </Tab>
  <Tab title="PayPal" icon="paypal">
    Enable PayPal Express Checkout.

    <CodeGroup tabs="JavaScript,Python">
      ```javascript
      // Initialize PayPal SDK
      paypal.Buttons({
        createOrder: function(data, actions) {
          return actions.order.create({
            purchase_units: [{
              amount: { value: '10.00' }
            }]
          });
        }
      }).render('#paypal-button-container');
      ```
      ```python
      import requests
      response = requests.post(
        'https://api.sandbox.paypal.com/v2/checkout/orders',
        auth=('CLIENT_ID', 'CLIENT_SECRET'),
        json={
          'intent': 'CAPTURE',
          'purchase_units': [{'amount': {'currency_code': 'USD', 'value': '10.00'}}]
        }
      )
      ```
    </CodeGroup>
  </Tab>
</Tabs>

## Social Media Login Options

Enable one-click logins to boost user sign-ups.

<Steps>
  <Step title="Enable Google OAuth" icon="google">
    Go to `https://dashboard.market360.com/auth/google` and enter your Client ID and Secret.
  </Step>
  <Step title="Add Facebook Login">
    Register your app in Facebook Developers and configure the redirect URI: `https://api.example.com/auth/facebook/callback`.
  </Step>
  <Step title="Test the Flow">
    Users will see social buttons on your login page. Handle the callback in your backend:

    ```javascript
    app.get('/auth/google/callback', (req, res) => {
      // Verify token with Google
      passport.authenticate('google', { successRedirect: '/', failureRedirect: '/login' })(req, res);
    });
    ```
  </Step>
</Steps>

## Webhook Setup for Custom Notifications

Set up webhooks to receive real-time events like order creations or payment confirmations.

### Register a Webhook Endpoint

Use the Market360 API to subscribe to events.

<Request tabs="cURL,JavaScript">
  ```bash
  curl -X POST https://api.example.com/v1/webhooks \
    -H "Authorization: Bearer YOUR_API_KEY" \
    -H "Content-Type: application/json" \
    -d '{
      "url": "https://your-webhook-url.com/market360",
      "events": ["order.created", "payment.succeeded"]
    }'
  ```
  ```javascript
  const response = await fetch('https://api.example.com/v1/webhooks', {
    method: 'POST',
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({
      url: 'https://your-webhook-url.com/market360',
      events: ['order.created', 'payment.succeeded']
    })
  });
  ```
</Request>

### Handle Incoming Webhooks

Verify signatures and process payloads.

<Response tabs="Success,Failure">
  ```json
  {
    "event": "order.created",
    "data": {
      "id": "ord_123456",
      "user_id": "usr_789",
      "total": 25000,
      "currency": "SLL"
    }
  }
  ```
  ```json
  {
    "error": "Invalid signature",
    "message": "Webhook verification failed"
  }
  ```
</Response>

<Expandable title="Advanced: Verify Webhook Signature" default-open="false">
  Compute HMAC SHA256 of the payload using your webhook secret.

  ```javascript
  const crypto = require('crypto');
  const signature = 'sha256=' + crypto.createHmac('sha256', 'YOUR_WEBHOOK_SECRET')
    .update(payload)
    .digest('hex');
  // Compare with header: req.headers['x-market360-signature']
  ```
</Expandable>
