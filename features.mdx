## Overview

Market360 empowers you to buy, sell, hire, and trade within Sierra Leone's vibrant digital economy. Tailored for local users, it combines intuitive tools for seamless transactions, secure payments via mobile money, and nationwide delivery options. Whether you're a buyer seeking fresh produce or a seller offering freelance graphic design, Market360 handles it all in one platform.

<Callout kind="info">
  All features support Leones (SLL) pricing and integrate with local providers like Orange Money and Afrimoney.
</Callout>

## Buy and Sell Products

Post your products effortlessly and reach buyers across Sierra Leone. Use high-quality photos, detailed descriptions, and competitive pricing to stand out.

### Quick Listing Steps

<Steps>
  <Step title="Create Listing" icon="plus">
    Navigate to the "Sell" tab and select your category, such as "Electronics" or "Agriculture".
  </Step>
  <Step title="Add Details" icon="edit">
    Upload photos, set price in SLL, and describe condition.
  </Step>
  <Step title="Publish" icon="upload">
    Review and publish. Buyers can message you instantly.
  </Step>
</Steps>

For developers integrating listings via API:

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  const response = await fetch('https://api.example.com/v1/listings', {
    method: 'POST',
    headers: { 'Authorization': 'Bearer YOUR_TOKEN', 'Content-Type': 'application/json' },
    body: JSON.stringify({
      title: 'Fresh Mangoes - 10kg',
      price: 50000,
      category: 'Agriculture',
      location: 'Freetown'
    })
  });
  ```
  ```python
  import requests
  response = requests.post(
      'https://api.example.com/v1/listings',
      headers={'Authorization': 'Bearer YOUR_TOKEN', 'Content-Type': 'application/json'},
      json={
          'title': 'Fresh Mangoes - 10kg',
          'price': 50000,
          'category': 'Agriculture',
          'location': 'Freetown'
      }
  )
  ```
</CodeGroup>

## Hire Services and Freelancers

Find skilled professionals for one-off tasks or ongoing projects. Rate providers post-job to build trust in the community.

<Tabs>
  <Tab title="Post a Job" icon="postcard">
    Specify requirements like "Logo design for my shop" with budget and deadline.
  </Tab>
  <Tab title="Browse Providers" icon="users">

  </Tab>
  <Tab title="Hire & Pay" icon="handshake">
    Chat, agree terms, and release payment upon completion.
  </Tab>
</Tabs>

<Callout kind="tip">

</Callout>

## Trade Goods Securely

Swap items without cash using Market360's escrow system. List what you offer, propose trades, and let the platform hold goods until mutual confirmation.

| Trade Type | Description | Security Feature |
|------------|-------------|------------------|
| Direct Swap | Exchange bikes for electronics | Escrow hold |
| Multi-Item | Trade phone + cash for laptop | Dispute mediation |
| Group Trade | Community bulk swaps | Verified participants |

## Localized Payments and Delivery

Handle transactions with Sierra Leone-specific options. Integrate MNO wallets for instant transfers and partner with local couriers for reliable delivery.

<Expandable title="Advanced Payment Integration" default-open="false">
  Configure webhooks for real-time notifications:

  ````javascript
  app.post('/webhook/payments', (req, res) => {
    if (req.body.status === 'completed') {
      // Fulfill order
      console.log('Payment from Orange Money confirmed');
    }
  });
  ````

  Set your webhook URL to `https://your-webhook-url.com/webhook`.
</Expandable>
